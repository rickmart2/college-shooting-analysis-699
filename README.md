# college-shooting-analysis-699

## Running the Code

To run the code, first reference 'requirements.txt' to see all libraries necessary for this analysis. Optionally, run 'college_data_preparation.ipynb' and 'nba_data_preparation.ipynb' to generate cleaned data (which is also already provided). From there, run 'analysis.ipynb', which will generate all relevant tables and visualizations for the project.

## Data Access Statement

NBA Data: All NBA shooting data are year-by-year tables downloaded from the Synergy Sports website. Login credentials are required to access this data. Advanced metrics are from Basketball Reference.

College Data: I manually compiled the college shooting data from a mix of sources. The only source that required login credentials was Synergy Sports. My other data sources, Basketball Reference and NBA.com, can be easily accessed online.
